# Minecraft Bot

A Minecraft bot that connects to a server with an intuitive graphical user interface (GUI).

## Features

- Create a new bot
- Connect and disconnect the bot

### Planned Features

- Set Minecraft server address
- Store server and bot usernames
- Chat functionality
